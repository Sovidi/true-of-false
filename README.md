# wonderplace_clone 😁


### 소개

원더플레이스, 첫 프로젝트 작품입니다.

원더플레이스의 브랜드 소개 페이지를 클론코딩하였습니다.

### 개발기간

> 2023.05.30 ~ 2023.06.15
> 

### 개발환경

> Windows 10
> 

> Visual Studio Code
> 

### 사용언어

> HTML
> 

> SCSS
> 

> Javascript
> 

### 플러그인

> Swiper
> 

### 특이점

- 기본적인 CSS와 HTML 을 시험하기 위한 첫 번째 프로젝트였습니다.
- 플랫디자인과 화이트 블랙 컬러를 사용한 모던과 심플이라는 테마를 눈여겨 보았고 채택하게 되었습니다.
